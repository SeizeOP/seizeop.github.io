ABOUT

This config is in the KDL format: https://kdl.dev

/-<node> comments out the following node, and // comments out a single line. Check the wiki for a full description of the configuration.

Note that for readablility purposes I will be using JSONC source code blocks, as opposed to having to install the kdl-mode emacs package.

CONFIGURATION

Autostarts

Programs to run when Niri starts for the first time.

Start the emacs daemon for better emacs load times.

spawn-at-startup "~/.local/bin/serv-emacs"

Start Waybar, a commonly used bar for Wayland compositors

spawn-at-startup "~/.local/bin/launch-waybar"

spawn-at-startup "nm-applet --indicator"
spawn-at-startup "systemctl --user start plasma-polkit-agent"

Set initial wallpaper

spawn-at-startup "hyprpaper"

Inputs

configure input devices 

input {
    keyboard {
        xkb {
            // You can set rules, model, layout, variant and options.
            // For more information, see xkeyboard-config(7).

            // For example:
            layout "us,en"
        }

Swap Capslock and Ctrl keys for convienence 

    options "grp:win_space_toggle,compose:ralt,ctrl:swapcaps"
}

Enable numlock on startup, omitting this setting disables it. 

    numlock
}

Touchpad

touchpad {
    // off
    tap
    // dwt
    // dwtp
    // drag false
    // drag-lock
    natural-scroll
    // accel-speed 0.2
    // accel-profile "flat"
    // scroll-method "two-finger"
    // disabled-on-external-mouse
}

Mouse

mouse {
        // off
        // natural-scroll
        // accel-speed 0.2
        // accel-profile "flat"
        // scroll-method "no-scroll"
    }

Trackpoint

trackpoint {
       // off
       // natural-scroll
        accel-speed 1.5
       // accel-profile "flat"
        scroll-method "on-button-down"
       // scroll-button 273
       // middle-emulation
   }

Focus on hover

  focus-follows-mouse max-scroll-amount="0%"
}

Outputs

output "eDP-1" {
    mode "1920x1080@60.000"
    scale 2
    transform "normal"
    position x=1280 y=0
}

Layout

  layout {
      gaps 12
      background-color "transparent"
      center-focused-column "on-overflow"
}

preset column widths

 preset-column-widths {
     proportion 0.33333
     proportion 0.5
     proportion 0.66667
 }

default-column-width { proportion 0.5; }
